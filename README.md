# Cultural Navigator

MPA-приложение на Spring Boot MVC для поиска культурных событий, площадок и пользовательских маршрутов.
Подробнее про технический стек, архитектуру, сущности, безопасность и т.д. можно прочитать в [требования.md](%D1%82%D1%80%D0%B5%D0%B1%D0%BE%D0%B2%D0%B0%D0%BD%D0%B8%D1%8F.md)

## Запуск через Docker

1. Скопируйте `.env.example` в `.env` и при необходимости заполните ключи Yandex API.
2. Соберите jar: `./gradlew bootJar`
3. Запустите контейнеры: `docker compose up --build`
4. Откройте `http://localhost:8080`

Администратор из seed-миграции: `admin / admin`.
